## Codebase facts (already verified by reading — don't re-explore)

- **Structure:** screens in `app/src/screens/`, primitives in `app/src/ui/` (PressableScale, PlantAvatar, Segmented, SelectRow, OptionSheet, DateSheet, Toast w/ undo, EmptyState, ScreenHeader, icons.tsx — Feather-style SVG set incl. Droplet, Check, Sprout, Clock, Bell), logic in `app/src/logic/` (schedule.ts = core algorithm, actions.ts = mutations returning undo tokens, notify.ts, fertilize.ts, profiles.ts ~350 entries, constants.ts), data in `app/src/data/db.ts`.
- **db.ts:** in-memory cache + Firestore + AsyncStorage mirror (`plantaroo:cache:{uid}`). Reads sync via `getPlants()/getHistory()`; ALL writes via `dbPut/dbAdd/dbDelete`; `subscribe()` powers `usePlants()/useHistory()/usePlant(id)` hooks in `app/src/ui/hooks.ts`. `startSession` hydrates disk then background cloud refresh. Integer ids via `genId()`.
- **Watering flow:** `useWaterAction()` in `app/src/ui/useWater.ts` (late Alert → `waterPlant` → undo toast → `rescheduleWateringReminders(getPlants())`). `actions.ts` also has `skipPlant`, `repotPlant`, `setLastWatered`, `bulkSetLastWatered`, `patchPlant`, `saveNotes`, `removeHistoryEntry`/`recomputeWateringState` (rebuilds schedule from history).
- **schedule.ts:** `getDaysUntilDue` returns -Infinity for never-watered; `getDueText` gives status `'never'|'overdue'|'today'|'soon'|'later'`; clamp 0.5–2× soil-adjusted baseline; seasonal multiplier display-time only.
- **Navigation:** native stack (SignIn / Tabs / PlantDetail / AddPlant / Settings / UpdateRequired) + 3 bottom tabs via custom `TabBar.tsx`. `navigationRef` in `app/src/navigation/ref.ts` for imperative nav (quick actions use it).
- **App.tsx:** auth gate, forced-update gate, quick-actions bridge, ToastProvider wraps NavigationContainer. Notification permission deliberately NOT requested at launch.
- **Photos:** `app/src/lib/photo.ts` — 400px JPEG q0.6 base64 data-URI stored in the plant doc.
- **Theme:** `app/src/theme.ts` — dark-only, tokens for colors/radius/spacing/font/shadows, `dueColor()` helper. Design rules: no pills, no emoji, SVG icons, weights ≤700, quiet.
- **Types:** `app/src/types.ts` — `Plant`, `HistoryEntry` (`type: 'Watered'|'Watered + Fed'|'Skipped'|'Repotted'`, `lateReason`), `Store = 'plants'|'history'|'profileCache'`.
- Gesture-handler v2.28 and Reanimated v4 installed; `GestureHandlerRootView` already at root. `@gorhom/bottom-sheet`, `expo-blur`, `expo-linear-gradient`, FlashList available.
